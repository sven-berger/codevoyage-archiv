Archiv von codevoyage.de

Hier findest du die verschiedenen Versionen von codevoyage.de, die ich im Laufe der Zeit entwickelt habe. Jede Version hat ihre eigenen Stärken und Schwächen.

1.0.0

  • Komplett entwickelt ohne Hilfen von LLMs
  • Kein reponsives Design
  • Keine Werkzeuge/Frameworks wie Bootstrap
  • Einfaches HTML und CSS
  • Kein JavaScript
  • Keine Objektorientierung beim Programmieren
  • Backend war überladen, da ich noch mit Subdomains arbeitete und jede Subdomain eigene Einbindungen benötigte

2.0.0

  • Frontend entwickelt ohne Hilfen von LLMs
  • Immer mehr Hilfe von LLM im Backend
  • Nur halb reponsives Design
  • Erstmaliger Einsatz von TinyMCE-Editor
  • Kein Einsatz von Werkzeugen/Frameworks
  • Erste Versuche mit Objektorientierunger Programmierung
  • Kein JavaScript
  • Erstes eigenes Login-System

3.0.0

  • Erstmaliger Einsatz von Bootstrap
  • Kein reponsives Design
  • Alte Skripte wurden nicht übernommen
  • Kein JavaScript
  • Version wurde recht schnell eingestellt, da JavaScript immer mehr in den Fokus geriet
  • Diente hauptsächlich zum Erlenrnen von Bootstrap
  • Backend wurde komplett neu entwickelt

4.0.0

  • PHP wurde dafür eingesetzt, wofür es da ist: Backend
  • Reponsives Design
  • Zum Einsatz kamen: Bootstrap und Stimulus
  • Frontend in JavaScript geschrieben
  • Erste Erfahrungen mit APIs, JSON und JavaScript
  • Keine Objektorientierung beim Programmieren
  • Es gab kein Backend
